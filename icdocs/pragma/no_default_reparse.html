no_default_reparse

Don't reparse the output from any of the Interchange container tags by default

Valid values

Valid values are 0 and 1.  The default is 0.

Description

Most container tags reparse their output by default.  This pragma disables this behaviour.

Reparsing can still be requested by setting the reparse attribute true when calling individual container tags.
