strip_white

Strip whitespace from the top of Interchange-served HTML pages.

Valid values

Valid values are 0 and 1.  The default is 0.

Description

This pragma strips whitespace from the top of HTML pages output by Interchange.  Such whitespace usually comes from Interchange tags which get processed and leave empty space behind them.  The purpose of this pragma is to move your HTML source closer to the top of a browser's "View source" display, and to use (slightly) less bandwidth when serving pages.
