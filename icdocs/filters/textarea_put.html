textarea_put

Replace every "&" character, in the body text, with a "&amp; HTML entity.  Also replaces "[" with "&#91;" and "<" with "&lt;"

Also see the textarea_get filter.

Example

[filter textarea_put]foo, [bar] & <baz>[/filter]

Results in:

Foo, &91;bar] & &lt;baz>

Source code

sub {
    my $val = shift;

    $val =~ s/\&/\&amp;/g;
    $val =~ s/\[/&#91;/g;
    $val =~ s/</&lt;/g;
    return $val;
}
