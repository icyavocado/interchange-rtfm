lc

Converts the given text into lower-case.

Also see the uc and lcfirst filters.

Example

[filter lc]Kevin Walsh[/filter]

Results in:

kevin walsh

Source code

sub {
    use locale;
    if ($Scratch->{mv_locale}) {
        POSIX::setlocale(LC_CTYPE, $Scratch->{mv_locale});
    }
    return lc(shift);
}
