sql

Change single-quote characters into doubled versions (' becomes '').

Also escapes blackslashes ("\" becomes "\\") unless you put the following somewhere on your page:

  • [pragma filter_sql_no_backslash 1]

Also see the "dbi_quote" filter.

Note

If you're using the [query] tag then you should use the [PREFIX-quote] sub-tag instead of this filter.

Example

[filter sql]Kevin's cat[/filter]

Results in:

Kevin''s cat

Source code

sub {
    my $val = shift;
    $val =~ s:':'':g;
    $val =~ s:\\:\\\\:g unless $::Pragma->{filter_sql_no_backslash};
    return $val;
}
