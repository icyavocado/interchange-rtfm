Looping tags and sub-tags

Certain tags are not standalone, but are dependent upon a parent tag.  The tags in this category are variously referred to as "looping tags", "sub-tags" or "pseudo-tags".  Looping tags include [item-list], [loop], [query], [search-region] and some others, such as [tree].

See the looping tags introduction for an explanation of how the various looping tags work.  The introduction also covers the various output control regions, such as [on-match], [no-match] and [more-list] etc.
