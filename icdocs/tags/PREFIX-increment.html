PREFIX-increment

Summary

  • [PREFIX-increment]

Description

The current loop iteration count.

The count starts from either 1 (in a zero-anchored list, such as [loop] or [item-list]) or from the match count in a search list.

Warning

If you skip items with [PREFIX-last] or [PREFIX-next] then the count will not be adjusted.
