PREFIX-modifier-name

Summary

  • [PREFIX-modifier-name attribute]

Description

For successive loop iterations, this sub-tag returns given "attribute" name followed by the loop's current iteration number (starting from zero), as follows:

  • attribute0
  • attribute1
  • attribute2

etc.

Note

[PREFIX-modifier-name quantity] is the same as [PREFIX-quantity-name], but that only holds true for the quantity attribute.

Warning

If you skip items with [PREFIX-last] or [PREFIX-next] then the count will not be adjusted.
