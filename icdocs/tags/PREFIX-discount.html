PREFIX-discount

Summary

  • [PREFIX-discount noformat]

Description

The price of the current item is calculated, and the difference between that price and the list price (quantity one) is output.

The result will be currency-formatted unless the optional "noformat" parameter is specified.  See the [currency] tag.

See the [discount] tag documentation for information on how to set and clear discounts.

Warning

This may have different results than you expect if you set the [discount] tag along with quantity pricing.  This tag returns the price difference for one item.  Multiplying the value with a [PREFIX-quantity] value will not take any CommonAdjust price breaks into account, so may not be a true reflection of the actual discount.
