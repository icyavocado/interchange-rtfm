Latest Interchange documentation

This Interchange documentation is the most accurate, reliable and up-to-date source of Interchange information you'll find anywhere.  Almost all of the 500+ pages of Interchange documentation text have been either created from scratch, rewritten or updated in some way.

All of the Interchange documentation pages are neatly categorised and extensively cross-referenced to help you quickly find the information you need In fact, the documentation is so good, this website even has its own awards section.

The Interchange documentation presented here is by no means complete.  If you discover some information that is present elsewhere, but missing from this website, then please let us know so that we can improve this resource for the benefit of all Interchange users.  Likewise, if you have any suggestions, or would like to contribute some text, then please get in touch.

I'm sure you'll find it refreshing to discover that contributed text will actually be used, if accurate, or corrected if obvious errors are found.  We find that it's better to publish as many contributions as possible, rather than ignore them or just let them pile up in a giant TODO list.

Interchange versions

The documentation on this website covers Interchange 5.0.0 through to the latest 0.0.0 (stable) and 0.0.0 (development) versions.  If a facility was removed prior to version 5.0.0 then it probably won't be documented here.

Documentation categories

  • Interchange tags
    Documentation for the Interchange markup language tags and their calling parameters. 

  • Interchange configuration
    Directives for use in the global (interchange.cfg) and/or local (catalog.cfg) configuration files, and in any files included from them. 

  • Interchange database
    Documentation for Interchange's database access facilities. 

  • Interchange forms
    Documentation for Interchange's form processing facilities. 

  • Interchange ecommerce functions
    Documentation for the ecommerce-related functionality provided by Interchange. 

  • Interchange Perl usage
    Pages associated with the use of Perl within Interchange. 

  • Data/text filters
    Data/text filters that can be used within Interchange pages and elsewhere.  See the Filter directive, and ICML tags such as [filter] and [PREFIX-filter] tags etc.
