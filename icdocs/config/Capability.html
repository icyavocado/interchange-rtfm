Capability

Load a module, if available, with no warnings if the module is not present.

Synopsis

Capability  type  name

Scope

This directive is only available for use in the global (interchange.cfg) configuration file, and will affect all websites running under the Interchange installation.  It will not work in a website's local (catalog.cfg) configuration file. 

Description

This is just like Require and Suggest, but will never cause a warning nor a message.  You can check for globalsub, sub, taggroup, usertag and module capabilities.

Example

The following would load the Archive::Zip module, if available.  A facility or extension, within Interchange, may check for that capability later and adapt itself accordingly.

Capability  module  Archive::Zip
