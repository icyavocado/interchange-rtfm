OrderReport

The location of the report template that will be emailed to the merchant when an order is placed.

Synopsis

OrderReport  filename

The parameter is expected to be a filename.  The default is "etc/report".

Scope

This directive is only available for use in the local (catalog.cfg) configuration file.  It will not affect any other website in any way.  This directive will not work in the global (interchange.cfg) configuration file. 

Description

The location of the report template that will be emailed to the merchant when an order is placed.  The file must be relative to the website's home directory, as specified with the global Catalog directive.

Note

If SendMailProgram is set to "none" then the report will not be emailed, and an error will be logged to the website's error log file.

Example

OrderReport  etc/order_email.merchant
