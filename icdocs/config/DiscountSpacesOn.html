DiscountSpacesOn

Enable the discount namespace switch feature.

Synopsis

DiscountSpacesOn  Yes

This is a Yes or No directive.  The default is No.

Scope

This directive is only available for use in the local (catalog.cfg) configuration file.  It will not affect any other website in any way.  This directive will not work in the global (interchange.cfg) configuration file. 

Availability

This directive was introduced in version 5.4.0, and is therefore not available for use with any earlier Interchange version.

Description

When DiscountSpacesOn is enabled, the current discount namespace may be switched using any of the following methods:

  • The [discount-space] tag,
  • The [discount] tag's discount_space parameter.
  • A CGI variable associated with the DiscountSpaceVar local configuration directive.

When this feature is not enabled (the default), only the "main" discount namespace will be available, and no namespace switch requests will be acted upon.  All attempt to change the discount namespace will result in an error in the website's ErrorFile.
