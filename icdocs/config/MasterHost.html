MasterHost

Synopsis

MasterHost  hostname_or_ip

The parameter is expected to be a regular expression listing one or more hostnames and/or IP addresses.

Scope

This directive is only available for use in the local (catalog.cfg) configuration file.  It will not affect any other website in any way.  This directive will not work in the global (interchange.cfg) configuration file. 

Description

This directive can be used to specify hostnames and/or IP addresses that page requests must match in order to be allowed access to directories protected by a non-zero-length ".access" file.  See the page access control documentation for more information.

Examples

IP address

MasterHost  127\.0\.0\.1

Hostanme or IP address

MasterHost  andromeda\.office\.cursor\.biz|81\.168\.51\.195
