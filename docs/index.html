Interchange documentation

   

This is the latest supporting documentation for the Interchange application server.


  • Interchange Tags Reference [1]
    List of most of the ICML tags and their calling parameters.  ICML, the Interchange Markup Language, is also known as ITL, IML, MML and various other weird acronyms.

    Last update: 20 July 2005.


  • Interchange Configuration [1]
    Application design, interchange.cfg, catalog.cfg, server administration.

    Last update: 05 May 2004.


  • Interchange Database [1]
    Source files, DBM, SQL, searching, sorting, shipping, userdb, order routing.

    Last update: 11 April 2004.


  • Interchange Templates [1]
    Variables, tag overview, programming, debugging, forms.

    Last update: 11 April 2004.


  • Interchange E-Commerce Functions [1]
    Ordering, pricing and CommonAdjust, taxing, the checkout process.

    Last update: 05 May 2004.


  • Interchange Foundation Demo Shop [1]
    Developing a shop with the demo application.

    Last update: 26 November 2002.


  • Interchange Catalogue-Building Tutorial [1]
    Guided tour showing how to build a very simple shop from scratch.

    Last update: 20 September 2004.


  • Interchange Internationalisation Features [2]
    Configuring and using Interchange's internationalisation (I18N) facilities.

    Last update: 30 October 2002.


  • Interchange Frequently Answered Questions [1]
    Frequently asked questions, with answers.

    Last update: 26 November 2003.


  • Interchange Advanced Topics [1]
    Maintaining Interchange, components, admin internals, link programs.

    Last update: 08 December 2003.


  • Interchange Table Editor [2]
    Reference guide for the Interchange table editor found in Interchange 4.9 and above.

    Last update: 29 December 2004.


  • Interchange UI Content Editor [1]
    Reference guide for the content editor in the found in the admin UI in Interchange 4.9 and above.

    Last update: 30 October 2002.


  • Interchange Forum Guide [4]
    Reference guide for the Forum feature found in the Foundation demo supplied with Interchange 4.9 and above.

    Last update: 26 November 2002.


  • Interchange Programmers' Reference [2]
    Reference guide for Interchange programmers or for anyone else who wants more detailed information about the Interchange internals and how it all works.

    Last update: 07 January 2004.


  • Interchange Special Variables [2]
    Interchange defines some special variables which control behavior.  These variables are documented here.

    Last update: 13 January 2004.


  • Interchange Quickbooks Integration [3]
    QuickBooks support for transactions and items.

    Last update: 03 July 2003.


  • Interchange CVS Guide [3]
    How to use CVS in your Interchange application development.

    Last update: 26 November 2003.


  • Interchange Upgrade Guide [1]
    Tips on moving from MiniVend 3 to 4, or from MiniVend 4 to Interchange.

    Last update: 26 November 2002.

Source code

The docuentation source code, in SDF format, is available from the Interchange CVS archive.  The SDF compiler can be used to build the Interchange documentation in various formats including HTML, PDF, POD, plain text and others.  You'll find instructions on how to connect to the Interchange CVS archive here.  Just remember to "checkout" the docs module, rather than the interchange module.  Why bother with all of that when you can just read the latest version here?
