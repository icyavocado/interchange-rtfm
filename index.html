Interchange user location map

Kevin Walsh - 13 February 2008 - 8:15 AM (EST) RTFM website news

Just for fun, this website now allows registered users to plot their location on a map.  A link to the map can be found in the navigation menu shown on the top-left of every page.  If you go to your account information page then you'll see a new "show me on the user map" option.  You can leave that set to "no", or you can set it to "yes" and plot your location on the map.  You can modify your location, or remove yourself from the map altogether, any time you like.
