New and replacement UserTags for Interchange

   

Note that local UserTags can be installed globally, but global UserTags will not work as expected when installed locally.

Global UserTags

 Name   Size   Description
count-sessions 1,886 Count the number of "active" sessions.

Author: Kevin Walsh.

Last update: 29 April 2002.
encrypt 2,093 Encryption of arbitrary data within Interchange pages for emailing and database storage etc.  Be careful - installing this tag will allow any Interchange page programmer to execute any command with the same access rights and permissions as the Interchange server's user ID.

Author: Ed Lafrance.

Last update: 23 March 2001.
email 1,051 Replacement for the email global usertag.  This version, taken from Interchange 4.9.0, allows the use of Net::SMTP.

Author: Red Hat.

Last update: 04 February 2002.
enable-downloads 2,811 Set up symlinks to allow softgoods to be downloaded securely.  This is the article that was posted to the interchange-users mail list.

Author: Mike Heins.

Last update: 28 February 2002.
env 1,778 Allows read access to the various HTTP environment variables.

Author: Ed LaFrance.

Last update: 29 January 2002.
image-size 651 Calculates the width and height of an image file and writes the values into the "imgheight" and "imgwidth" scratch variables.

Author: Jonathan Clark.

Last update: 13 June 2002.

Local UserTags

 Name   Size   Description
generate-rand 1,336 Generate string of random numbers, for possible use as some sort of "key".

Author: Kevin Walsh.

Last update: 29 April 2002.
is-secure 1,192 Check whether the current page has been displayed using the SSL protocol.

Author: Kevin Walsh.

Last update: 29 April 2002.
lookup-qty-pricing 3,376 Return the correct price for an item based on the quantity being ordered.  Uses a custom quantity pricing table to define the price breaks for each item.  This is the article that was posted to the interchange-users mail list.

Author: Ron Phipps.

Last update: 19 April 2002.
subtotal-nodisc 1,550 Shows the current cart subtotal before any discounts are applied.

Author: Kevin Walsh.

Last update: 29 April 2002.
