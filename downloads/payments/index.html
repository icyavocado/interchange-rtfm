Interchange payment modules

   

These are the latest versions of the Interchange payment modules.  Please note that these modules come from the development release, so any module with an "update date" later than the latest Interchange version may have problems.  Please report any problems.

If you have a working payment module that is not listed here then please donate it.  If you know of another payment service that can make use of one or more of the modules listed below then please let us know.  If a seemingly useful installation or configuration guide for any of these modules is posted to the interchange-users mail list, the Interchange BBS or elsewhere, we will probably put a link to it on this page.

Check if you have the module already before you download it from here. The only reasons to download from here are either (1) you don't have the module or (2) you are having problems and we have a later version.  Don't update for the sake of it; if it's not broken, don't fix it.
