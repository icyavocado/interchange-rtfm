HOWTOs and other Interchange documentation

   

Note that other mini HOWTOs can be found on the Interesting Mail List Articles page.


Interchange + Excel HOWTO A quick HOWTO on uploading and downloading Excel-format files using the Interchange admin UI.

Author: Dan Browning

Last known update: 26 July 2002.  (External page)

Interchange + FreeBSD + PostgreSQL HOWTO A HOWTO on installing Interchange 5.2.0 on a FreeBSD server and connecting to a PosstgreSQL database.

Author: W3 Trader, Inc. (www.w3trader.com).

Last known update: 21 September 2004.  (External page)

Interchange setup for the UK This document explains how to alter the default Interchange 'foundation' template catalogue to:
  • Display prices in UK Pounds Sterling by default.
  • Enable Euros for alternative pricing.
  • Enable VAT sales tax.
  • Add Switch and Delta cards to the credit card payment method.

Author: Jonathan Clark (www.webmaint.net).

Last known update: 19 February 2003.  (External page)

Introduction to online credit card processing (123k PDF). A description of the steps needed to set-up credit card processing for online shops.  This document is not Interchange-specific and should be of interest to anyone who needs to accept credit card payments online.

Author: Marc Brevoort

Last update: 15 January 2003.
Minimum and batch quantities   A guide to implementing batch and minimum quantities to the products in your shop.

Author: Guido Sohne

Last update: 18 September 2001.
Foundation Modification HOWTO Instructions on how to modify the Foundation demo, the Interchange admin interface and the database to add a new product display field.

Author: W3 Trader, Inc. (www.w3trader.com).

Last known update: 13 October 2004.  (External page)

PayPal payments   Another PayPal article.  This code was posted to the interchange-users mail list in this article.

Author: Scott Kennedy (custom-web-promotions.com).

Last update: 12 May 2002.
Piled interchange remarks Lots of useful Interchange comments, gathered together in one place.
(Same document, but in German: Wissenwertes über Interchange.)

Author: Stefan Hornburg (www.linuxia.de).

Last known update: 26 March 2002.  (External page)
Preparing your materials for Interchange This documentation consists of a PDF file and some .xls files given to Currant Media clients to help them prepare their product information and images for upload to a Foundation-style Interchange e-commerce web site.  Julia Jabobs of Currant Media has given her kind permission for this excellent documentation to be shared via this site.

Author: Julia Jacobs (www.currantmedia.com).

Last known update: 24 April 2002.  (External page)

Remote procedure calls with SOAP   A guide to calling Interchange's tags and facilities, using the SOAP::Lite module, while maintaining an Interchange session.  Calls from Interchange's [soap] tag to remote SOAP servers is not covered by this document.  See the Interchange Tags Reference in the Documentation section of this site.  Once you have read this HOWTO, the [soap] entry in the ictags document will make sense.

Author: Mike Heins (www.perusion.com).

Last update: 25 February 2001.

Donations
